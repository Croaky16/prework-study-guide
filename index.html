HTML

  • Enter your HTML notes here
  • HTML is used to create the structure of a webpage.
  • An element is a type of content on a webpage that work together to build the structure of a webpage and provide a framework that makes it easy for a user to navigate the webpage.
  • The html tag represents the root/start of the HTML document.
  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • The title element defines the title of the webpage.
  • The base element defines the base URL for the webpage.
  • The link element connects an external resource to the HTML document.
  • The meta element defines the metadata such as the character set, description, keywords, author, and viewport.

CSS

  • Inline CSS is applied to a specific HTML tag to style that particular page element only.
  • Internal Style Sheet contains CSS rules for the webpage in the head section of the HTML file.
  • External CSS style sheet allows developers to keep all of their CSS rules in a separate file.
  • We link an external style sheet through the link element in the HTML's head.
  • declarations contain the CSS property we want to apply and the value of the property.
  • A margin property indicates how much space we want around the outside of an element.
  • The padding property adds space around the content inside an element.

Git

  • Git is a version control system that allows you to manage and keep track of each version of your project.
  • git revert: turns the code back to the specified version control.
  • git status: shows modified files in working directory, staged for your next commit.
  • git commit -m: commits your staged content as a new commit snapshot.
  • git clone: retrieves an entire repository from a hosted location via URL.

JavaScript

  • A variable is simply a named container that allows us to store data in our code.
  • The const keyword allows us to create variables that can't be reassigned a value.
  • We use an assignment operator = to assign the specific data that we want stored in our variable.
  • All strings are surrounded by quotation marks.
  • Control flow is the order in which a computer executes code in a script.
  • We can use conditional statements and loops to control the order in which code is executed, or if it is even executed at all.
  • An array is a single variable that is used to hold a group of data. Arrays are typically used to hold data that is related in some way.
  • Arrays are not limited to contain only a single type of data. A single array can store different types of data, such as strings, numbers, and so on.
  • In JavaScript and many other programming languages, arrays are zero-indexed and sequential.
